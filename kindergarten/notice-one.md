## 幼儿园注意事项一

### 家长觉得刚入园宝宝在幼儿园吃饭慢、吃不饱怎么办?
先向老师了解孩子吃饭慢的原因。如果是因为在家都是成人喂饭而造成孩子吃饭慢的话，家长在家就要有意识地培养孩子自己吃饭；如果是因为家中给孩子做食物做得比较软，孩子的咀嚼能力比较弱而造成孩子吃饭慢的话，家长在家就要逐步不给孩子做过细过软的食物；如果是因为孩子总是边玩边吃的话，就要和老师一起慢慢地培养孩子集中注意力吃饭。
### 孩子在幼儿园喝水少怎么办?
> *  先通过老师了解孩子一天喝水情况，再观察孩子小便情况。如果小便量和颜色正常，家长则不必担心；如果小便量少且颜色不正常，家长则应与老师沟通配合解决。
> *  幼儿园老师一般都会提醒和鼓励孩子多喝水。
> *  如果孩子在幼儿园不爱喝白开水，家长应在家培养孩子自主喝白开水的习惯（糖和碳酸饮料饮用过多均不利孩子健康成长）。
### 刚入园宝宝在幼儿园不肯睡觉怎么办?
> * 午睡时间虽然短暂，但它可以帮助孩子缓解疲劳，同时睡眠时孩子的脑垂体会分泌生长激素，促进儿童长身体，还能满足孩子充足睡眠的需要，起到增强机体防护功能的作用。
> *  首先要逐步改掉抱着、哄着或是喝奶才能入睡、以及与成年人同睡一张床的习惯，告诉宝宝只有小小孩才需要那样睡觉，要上幼儿园的宝宝已经是大孩子了，鼓励宝宝像大人一样，自己躺在床上、盖好被子睡觉。
> *  如果你的宝宝的确有入睡困难的问题，或是在家里就没有午睡的习惯等，需要老师给予特别照顾，入园时一定要提前和老师沟通，让老师有所准备，老师会在入园初期像妈妈一样陪着孩子。
> *  培养孩子早睡早起的习惯。家长不要陪孩子玩得太晚、太兴奋；睡前洗个热水澡，有利于孩子入睡；与老师配合，鼓励孩子在园多运动、多锻炼。
> *  如果幼儿园允许，爸爸妈妈也可以给宝宝带一个在家里睡觉时习惯抱在怀里的毛绒玩具，或给宝宝带一个自己的小被子，熟悉的感觉和味道也能帮助宝宝有安全感而尽快入睡。
### 孩子在幼儿园常尿裤子怎么办?
> * 首先，3岁孩子偶尔出现尿裤子是正常现象，家长要正确对待。
> * 其次，与老师一起分析孩子尿湿裤子的原因。如果是因为贪玩没有来得及去厕所而尿裤子的话，要与教师沟通，多让老师提醒他去厕所。并且可以给孩子多带一些衣裤。这样即使尿了裤子，教师也可以及时给他换。如果是因为紧张而尿裤子，要和老师协调好，帮助孩子找到紧张的原因，消除孩子紧张的因素，慢慢改掉尿裤子的习惯。
### 孩子在幼儿园不爱大便、便秘怎么办?
> *  孩子不爱在幼儿园大便有时是因为不会擦屁股，有时是因为不习惯蹲坑，有时是因为在幼儿园大便会害羞。家长要与老师一起找到孩子不在幼儿园大便的原因，帮助幼儿逐步适应在幼儿园大便；同时要告诉幼儿憋着大便会对身体有伤害，很容易生病。
> *  家长要让宝宝的生活有规律，尽量帮助宝宝养成在早晨起床后按时排便的习惯。如果没有正常排便习惯，入园时及时与班级老师说明情况，以便老师关注。
> *  调整好宝宝的饮食，粗细搭配、营养均衡，尤其是纠正宝宝不爱吃蔬菜的习惯。
> *  宝宝便秘如果是入园焦虑所致，就要想办法安慰他，消除他刚入园的紧张心理。
> *  请老师多关心提醒孩子，告诉孩子有便意时候一定不能忍着，否则自己不舒服还容易生病。
⑤家长可以利用外出上公厕的机会，适当锻炼孩子蹲坑的能力。
### 新入园宝宝爱生病怎么办?
> *  多给宝宝参加户外活动的机会，锻炼身体，增强体质。
（2 ）家园合作鼓励孩子多喝水、适当地补充营养食物。
> *  如果孩子生病了，最好能等病好后再去幼儿园，避免交叉感染，更有利于孩子的健康。
> *  家长要关注天气预报，随着天气变化适时为宝宝增减衣服。
### 宝宝不按时起床怎么办?
> *  和孩子玩穿衣服比赛。
> *  和孩子一起设个小闹钟。
> *  及时和老师沟通，家园同步进行监督。
### 回家后应该给宝宝吃些什么?
> *  接孩子回家后，可以给他准备一盒牛奶或酸奶。如果幼儿园不供应牛奶，最好早上或者睡前让孩子喝一次奶。
> *  回家后，可以为孩子准备一些新鲜的水果作为零食，多喝白开水。
> *  幼儿园晚餐的进餐时间一般是下午16:30左右，而家庭晚餐时间是18:00～19:00，就算孩子在幼儿园吃了晚饭，也会跟着家长再吃一些，这是正常的，只是最好让孩子吃一些易消化的食物。
### 孩子不爱上幼儿园怎么办?
> *  仔细聆听孩子的心声，给予理解。
> *  家长要正面引导孩子。客观地、认真地给孩子讲解幼儿园好玩的游戏和有小伙伴的快乐生活。
> *  消除孩子的入园焦虑。耐心安抚孩子的情绪，多启发幼儿：幼儿园的玩具多、朋友多、老师好。告诉孩子会准时来接他回家。
> *  每天把孩子从幼儿园接出来后，可以给他一些和别的孩子接触的机会，帮助孩子交朋友；和老师交流，让孩子得到老师的关注。
> *  切忌语言威胁恐吓孩子。
### 放学接宝宝回家有什么技巧?
> *  和宝宝讲明去幼儿园的道理和条件后，在放学时准时去接宝宝，是打消顾虑、建立信任、形成积极心态的重要环节。
> *  如果万一不能按时来接孩子，一定及时通知老师，请老师在放学后特别关注宝宝；家长见到宝宝后，也要认真地向宝宝解释来晚的原因，向他道歉并向老师表示感谢。
> *  接宝宝回家时注意自己心态。宝宝刚刚上幼儿园，爸爸妈妈也一样会心神不宁、担心紧张。大人过多的焦虑情绪，会传递给宝宝一个信息：“你在幼儿园一天，我们也担心的不得了，也是一百个不放心。”这种焦虑和消极的态度对宝宝的情绪也会带来不利的影响，所以即使妈妈再担心、不安，也要表现出很踏实、很轻松的样子。
### 离园后，我们怎样询问宝宝一日在园情况?
> * 父母每天接孩子回家时，是孩子思维最活跃、最兴奋的阶段，也是培养幼儿语言表达能力和思维灵活性的黄金时间。父母可以好好地利用这一段时间，如提一些这样的问题：
> * 你今天最快乐的事是什么？为什么它是最快乐的事？今天你和小朋友一起做了什么游戏？你听见了（看见了）哪些有趣的事？你的好朋友是谁?如果你想帮助小朋友，最想帮助谁，为什么？
### 怎样询问“今天学了什么”?
> * 家长回家可以和孩子们玩一个游戏，让孩子当老师，家长当小朋友，这样家长可以在游戏中了解孩子在幼儿园一天的活动。
### 宝宝在幼儿园“打”人怎么办?
> *  对孩子的行为要及时向老师询问清楚。
> *  对孩子的行为要向被“打”的家长和孩子道歉。
> *  分析孩子“打”人的原因。
> *  日常生活中教育孩子礼貌待人，学习基本的礼貌用语，要让孩子懂得分享、懂得爱护和照顾别人。
> *  要教给宝宝与人交往的正确方法及自己处理问题的方法。
### 宝宝在幼儿园被小朋友“欺负”怎么办?
> *  询问孩子情况。
> *  及时向老师了解情况。
> *  安抚孩子，让他知道受到“欺负”并不是他的过错，“欺负”别人的行为是错误的；同时告诉孩子妈妈（爸爸）会帮助他。
> *  培养孩子的交往技巧。
> *  教会孩子自我保护的方法。
### 宝宝把幼儿园的东西带回家怎么办?
> *  不要用成人的眼光看待这个行为。
> *  要分析孩子的行为出于什么心理。
> *  要让孩子明白不经过别人同意而拿别人的东西是不对的。
> *  要及时和老师沟通说明孩子的行为。
### 宝宝说“老师吓唬我”怎么办?
> * 对孩子说的话不能不信也不能全信，因为孩子们的理解和表述毕竟和成人有所差别。正面引导孩子，让孩子知道他的某种行为让老师很生气；同时也和老师交流，让老师了解自己的某种行为对宝宝可能会造成一些影响，和老师一起找一找适合宝宝的教育方法。
### 宝宝在幼儿园受了委屈怎么办?
> *  了解孩子在幼儿园受了什么样的委屈，一边听一边帮孩子分析并开导他。
> *  正面帮孩子分析事件，让孩子知道以后再遇到这样的事怎样解决；应教给孩子一些处理类似事情的小方法。
### 家长与老师沟通的方式有哪些?
> *  面对面沟通：面对面沟通具有灵活、直接，方便进行个别家长与教师的沟通。幼儿园可通过离园、家访、约谈等形式，与家长进行幼儿情况的沟通。
> *  微信：微信是比较便捷的一种沟通方式。老师可以通过图像、文字等让家长及时了解到孩子在幼儿园的情况。家长也可以通过微信探讨育儿方法，交流自己的教育想法。
> *  互动研讨：围绕家长关注的话题，通过幼儿园或者班级专题研讨、家长沙龙等形式，在家园互动中传播理念，分享经验，提高家长科学育儿水平，从而增进家园情感，达成教育共识。
### 听到教师指出自己孩子的“不足”后，家长该怎么办?
> *  客观观察孩子的情况，准确了解孩子“不足”的具体表现、行为。
> *  与教师探讨或者搜集相关问题解决办法帮助孩子改善。
### 家长在教育问题上与教师有不同观点，应如何交流心平气和地和老师交流?
> * 一方面把自己的观点和老师聊聊，另一方面认真地听听老师的意见。客观地看待自己的孩子，分析存在哪些不足、怎么造成的。尝试用教师建议的方法或其他教育方法培养孩子。
### 宝宝在园发生问题，教师未能及时发现和处理，家长该怎么办?
> *  用平和的心态和老师沟通，耐心听一听老师的解释。
> *  教孩子一些自我保护的方法和以后再发生类似事件的解决办法。
### 孩子在幼儿园发生了意外怎么办?
> *  根据幼儿在园受伤程度，幼儿园会启动不同级别的应急预案及处理流程。一般情况下，教师会先将孩子带到保健室，由保健医生决定应如何处理。如果需要去医院，保健医生和带班教师会带孩子去医院。
> *  幼儿园会第一时间通知家长孩子受伤情况，家长可一起随行去医院。
> *  带班老师会客观地讲述孩子的受伤经过，让家长了解情况，并予以理解。
> *  如果属于责任事故，可与老师沟通处理方式；如果对老师的处理结果不满意，可以向幼儿园领导反映情况，按照幼儿园相应制度处理。
### 幼儿常见病流行期应采取什么样的措施?
> *  流行病期间，幼儿不混班、不串班、不办理入园、转园手续，所在班级要彻底消毒。
> *  医务室备有简单的意外急症抢救措施，并设有隔离室；若在园内发现患有传染病的幼儿，须立即送到隔离室进行隔离。
> *  患传染病的幼儿痊愈并隔离期满后，须持医院开具的复课诊断证明，方可返园。
> *  多带孩子进行户外运动，加强体育锻炼，增强体质，提高抗病能力。
家庭
> *  患有流行病或传染病的幼儿应在家调养休息，待病愈后方能返园。
> *  有条件的家庭在幼儿园发生流行病、传染病期间，可以选择在家看护自己的孩子。
### 家庭成员间教育意见不一致怎么办?
> *  孩子在场时要尽量避免正面冲突。
> *  寻找合适的机会以建议方式和家庭其他成员交流教育孩子的想法，确定有利于孩子发展的好方法，使家庭成员间教育意见尽量达成统一。
